Smart Drip Irrigation

IOT
Android
Python


        Basic principle behind our idea : The principle behind drip irrigation seeks efficiency of water usage by directing water to the base of each plant to maintain soil moisture content as per the plant’s needs.

The functioning will be as follows :

  • Automate the process of changing the gate valves through mobile app.
  • Can water the field depending on crop’s requirement.
The application consists of two modes of operation
  • Auto mode
  • Manual mode

        In Manual mode User inputs will be taken from android app having two buttons on/off which allows the user to open or close the gate valve.It also asks the user regarding type of crop.User’s input will be stored in the external server (000 webhost) .The information stored in the server is to be sent to Raspberri Pi where necessary action is to be taken. In auto mode the gate valve is opened by taking actual moisture level in the fields by reading moisture values by using sensor(YL-69) and opening a particular valves based on its readings

This can be implemented in 2 ways:
  • Can be integrated with existing drip irrigation
  • New drip irrigation system
