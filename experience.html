Accolite

  • I worked for Accolite for one and half year from January 2017 - Mau 2018. My development work here has ranged different technology stacks like NodeJs and AngularJs, Spring AngularJs, Java, Android App Development etc. Working for this company gave me a kick start into developing full-stack web applications with proper well defined architecture. The way they design an application so as to make it scalable, readable, flexible inspired me to follow such good practices.

    Here are the projects that I worked on during my journey at Accolite.

    •         FedEx FEM : FEM stands for Fatigue Event Management. This web application allows pilots and the management to communicate effectively on and off their journey. Every minute details right from sleep hours of pilots to flight delays and extended layover times are noted, analysed and statistically drawn to handle any kind of fatigue effectively in minimum time.

      Technology Stack : AngularJs 1.6, Spring MVC, SQL


    •         Cambium Networks (cn-maestro): This is also a web application designed for Cambium Networks for managing their networking devices like towers, access points, routers, wifis etc. remotely. I was involved only on the AngularJs (Front end) side of this application. We used AngularJs 1.6 in designing this application.

      Technology Stack : AngularJs 1.6, NodeJs, Postgresql


    •         Campus Select : This is an android application that helps hiring department while they visit any campus. This app will be installed by all the applicants or students and each student will be given an id. Entire process tracking can be done through this app like the current interview level of the student, venue of next round, hiring team details etc. On the other end, hiring team can update all the info and notify the students through an web portal which is made using AngularJs with SpringBoot backend. In this project :

      • I leaded the team of android
      • Contributed to a major extent in Spring Boot part of the application.
      • I also solely built a middleware for routing the requests through an aunthenticated channel.

      Technology Stack : Android, AngularJs 1.6, Spring MVC, SQL
